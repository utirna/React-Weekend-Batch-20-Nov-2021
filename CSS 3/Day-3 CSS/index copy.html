Google Web Designer is a drag-and-drop page builder for Windows, Mac and Linux from Google for creating interactive HTML5 ads and other HTML5 content. It offers a GUI with common design tools, such as a Text tool that integrates with Google Web Fonts, a Shapes tool, a Pen tool, and 3D tools.
